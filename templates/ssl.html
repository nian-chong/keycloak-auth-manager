📌 申请说明与注意事项
  • HTTP 验证：程序会自动在您的网站根目录下生成验证文件。请确保您的域名已经解析至本服务器，且防火墙已放行 80/443 端口。
  • DNS 验证：适用于没有开启 80 端口，或者需要申请泛域名证书的情况。需提前在 1Panel 的「证书」-「DNS 账户」中添加提供商 API 信息。
  • 自动重载：证书申请成功后，1Panel 会自动将证书绑定到对应的 OpenResty 站点，并自动重载 OpenResty 使其生效。
